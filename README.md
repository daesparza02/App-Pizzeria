# Pizzería Pai Pai 

Proyecto hecho en grupo para la asignatura de Ingeniería del Software II en el segundo año del Doble Grado en Ingeniería Informática y Matemáticas.

El proyecto consiste en la creación de una aplicación funcional para la gestión de una pizzería ficticia. El desarrollo se estructuró de forma modular, donde cada integrante se especializó en un subsistema concreto dentro de la app.
